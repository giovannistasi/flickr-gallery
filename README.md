## Getting started

**1.** Clone the repo

````
$ git clone https://github.com/jportella93/flickr-gallery.git
$ cd flickr-gallery
````

**2.** Install dependencies:

```
$ yarn install
```

or

    $ npm i

**3.** Go to /src/config/api.config.js and make sure that line 1 (dev) is active and line 2 (prod) is commented out.

**4.** Start the server

```
$ yarn start
```

or

    $ npm start
